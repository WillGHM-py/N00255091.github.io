How to use

In order to use the file reader, you need to a select choose a file, once selected find the file located named Exercises.csv then select it and confirm. It will then generate a table with all the values with a search bar with a drop down menu where you select the column and input the value to filter

Make sure the correct file is chosen, if not seen it may be in the downloads folder
